## [0.0.1] - 2026-03-01

### Added
- Initial release of Triforge AI Code Council
- Tri-model consensus engine: OpenAI GPT, Anthropic Claude, and xAI Grok
- Structured debate pipeline: Fast Draft → Risk Check → Critique → Consensus vote
- Right-click context menu: Explain Code, Write Tests, Refactor Code, Find Bugs
- Command palette: Add/Update/Remove API keys, Check Provider Status, Export Council Report
- Guided and Professional UI modes
- Configurable debate intensity and max iterations
- Export full AI debate log as Markdown
- Status bar indicator showing active council mode
- `Ctrl+Shift+T` / `Cmd+Shift+T` keybinding to open the panel
