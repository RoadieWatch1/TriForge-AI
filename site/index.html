How It Works

Up and running
in minutes

No technical setup. No cloud accounts to manage. Just download, add your API keys, and start thinking bigger.

1

Download & Install

Download the DMG for Mac (Apple Silicon). Drag TriForge AI into your Applications folder. Launch it. The splash screen greets you with a voice introduction while the app initializes.

2

Grant Permissions

A one-time setup wizard walks you through enabling the features you want — file access, printer control, voice input. You're always in control of what TriForge can do.

3

Add Your API Keys

Go to Settings and paste in your API keys from OpenAI, Anthropic, and/or Google. Keys are stored locally on your machine using encrypted storage — they never leave your computer.

4

Start Thinking Together

With all three models active, type your question. TriForge runs it through every AI simultaneously and delivers a synthesized, consensus answer. Your personal think tank is live.
